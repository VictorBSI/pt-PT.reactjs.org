### Como posso escrever comentários em JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* O comentário é escrito aqui */}
  Olá, {nome}!
</div>
```

```jsx
<div>
  {/* Também funciona com
  comentários de várias linhas. */}
  Olá, {nome}! 
</div>
```
